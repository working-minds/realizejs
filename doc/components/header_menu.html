Header Menu

Componente de cabeçalho de página.

  • Props
  • items Array
  • leftIcon String
  • rightIcon String
  • text String
  • href String
  • ref_id String

Props

items Array

Itens do menu.

default: []

leftIcon String

Nome do ícone do material-icons apresentado à esquerda.

default: ''

rightIcon String

Nome do ícone do material-icons apresentado à direita.

default: ''

text String

Nome do menu

default: undefined

href String

Endereço do link do menu.

default: undefined

ref_id String

Id de referência do menu.

default: 'headerMenu'
