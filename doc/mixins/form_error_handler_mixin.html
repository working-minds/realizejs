    Form Error Handler

    Mixin para criação de formulários de erro.

    • Props
    • errorMessage String
    • baseErrorParam String
    • onError Function
    • mapping Boolean

    Props

    errorMessage String

    Mensagem de erro mostrada ao usuário.

    default: 'Por favor, verifique o(s) erro(s) abaixo.'

    baseErrorParam String

    Objeto contendo os erros base.

    default: 'base'

    onError Function

    Função callback executada ao inserir um dado errado no Form.

    default: function(xhr, status, error) {return true; }

    mapping Boolean

    Caso true, mapeia as mensagens de erro da Input.

    default: true
