    Header Section

    Component to define a section of the header component with alignment configuration.

    • Props
    • align String
    • id String
    • className String

    Props

    align String

    Alignment of the component group of children header.

    default: 'left'

    id String

    Id of the element HTML component.

    default: ''

    className String

    Classes of the header section.

    default: 'hide-on-med-and-down'
